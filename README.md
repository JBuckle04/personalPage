# personalPage

My personal website built with Astro.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Customize

Update `/tmp/workspace/JBuckle04/personalPage/src/pages/index.astro` with your own details:
- Replace `YOUR_NAME_HERE` in the title and heading.
- Replace `YOUR_GITHUB_USERNAME` in the GitHub profile link.
